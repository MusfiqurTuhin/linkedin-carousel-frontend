## Deployment

### Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import the repository
4. Add environment variable:
   - `NEXT_PUBLIC_API_URL` = Your Railway API URL
5. Deploy!

## Local Development

```bash
npm install
npm run dev
```
